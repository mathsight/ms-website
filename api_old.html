MathSight API

Information for developers

Using our API

This API makes it possible for you to use our analytics data within your dashboard.

We are working with marketing agencies, toolbox providers and small to medium enterprises to create an extremely insightful set of metrics.
You are not allowed to white-label our data without our specific agreement. If you plan to show our data input to a third party- please don't do so without a prior discussion of the usage rules with us.

Parameters used in our API function

1. domainUrl= this will be a domain that you want the analytics data for

2. metrics= the type of data you would like in your response. 
This can be one or more of:
feature
Feature data from this domain's html pages		 statseo
 StatSEO metric based on amalgamated feature and Google search page view data and feature data over last 1, 3, 6, and 12 months		 highlight
 Highlighted features contributing to statseo
3. sort= the order in which you would like you results, can be one of the following:
url / metric
4. max-results= the number of results you would like to have returned.
5. offset= starting from what position results should be returned.
6. output= format of returning results. Available variants: JSON, CSV.

Our API enables you to get the following:

1. Html structural and feature data from your chosen domain.

2. Our StatSEO traffic metric figure - a measure of attractiveness to search visitors.
3. The structural and content features that are contributing most and least to your pages' traffic performance.

1. HTML structural and content feature data

Per request cost: £0.01

Query example:

{
[
 "https://api.mathsight.org/v3/statseo
?
domainUrl="www.example.com"
&metrics=feature
&sort=url
&output=json
&max-results=1000
&offset=0
]
}

2. StatSEO metric

Per request cost: £0.02, with Google Analytics access for this domain £0.01

Query example:

{
[
 "https://api.mathsight.org/v3/statseo
?
domainUrl="www.example.com"
&metrics=statseo
&sort=url
&output=json
&max-results=1000
&offset=0
]
}
